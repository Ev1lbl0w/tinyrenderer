# tinyrenderer

My version of a [tinyrenderer](https://github.com/ssloy/tinyrenderer/wiki), written in Dlang.

## Dependencies

- [dlang](https://dlang.org/)
- [dub](https://code.dlang.org/)

## Compilling

`dub build`

## Running

`dub run`, or `./tinyrenderer` after compilling

## Testing

`dub test`
